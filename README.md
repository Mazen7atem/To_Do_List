# To Do List App (Frontend-Only Version)

A simple and responsive To-Do List application built with HTML, CSS, and JavaScript (or React). No backend requiredâ€”tasks are saved in your browser using localStorage.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed or pending
- Filter tasks (All / Completed / Pending)
- Tasks are saved in browser (localStorage)
- Responsive design (optional)

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript (or React)
- **Backend:** None
- **Database:** Browser localStorage

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/To_Do_List.git
   ```
2. Navigate to the project directory:
   ```bash
   cd To_Do_List
   ```

## Usage
Simply open the `index.html` file in your web browser, or if using React, run:
```bash
npm install
npm start
```
Then visit `http://localhost:3000` in your browser.
